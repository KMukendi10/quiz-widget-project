# Project Overview

This project is an interactive JavaScript Quiz Widget built using:

- HTML
- CSS
- JavaScript

The quiz allows users to:

- Answer multiple-choice questions
- Navigate through questions
- Track score automatically
- View final results
- Restart the quiz
- Experience smooth animations and UI effects
